# Grupa7_TPTP_projekat
# Apex Movement – Sportska Oprema
## TPTP Završni projekat 2025/2026

Ovaj repozitorij sadrži završni projekat iz predmeta **Tehnologije za podršku tehničkom pisanju (TPTP)** na akademskoj godini 2025/2026. Projekat predstavlja web stranicu za prodavnicu sportske opreme "Apex Movement".

## Članovi tima i uloge
U izradi projekta učestvovala su tri studenta, podijeljena:

* **Ajdin Mehaković** - HTML (Struktura stranica, semantika, navigacija)
* **Ajla Haskić** - CSS (Dizajn, layout, responzivnost, Dark/Light mode stilovi)
* **Ajdin Mehmedović** - JavaScript (Validacija forme, filtriranje sadržaja, interaktivne komponente)

## Korištene tehnologije
* **HTML5**: Semantički ispravna struktura.
* **CSS3**: Napredni layout (Flexbox/Grid) bez eksternih frameworka.
* **JavaScript (Vanilla JS)**: Dinamičke funkcionalnosti bez eksternih biblioteka.
* **Google Fonts**: Barlow i Barlow Condensed fontovi.

## Ključne funkcionalnosti
1.  **Dark / Light Mode**: Mogućnost promjene teme koja se čuva u `localStorage`.
2.  **Filtriranje sadržaja**: Dinamičko filtriranje artikala na stranici `sadrzaj.html`.
3.  **Validacija kontakt forme**: Klijentska validacija rađena isključivo putem JS-a.
4.  **Interaktivni elementi**: Animirani brojači i glatki skrol (Smooth scroll).
5.  **Responzivan dizajn**: Prilagođeno za mobilne telefone, tablete i desktop računare.

##  Struktura repozitorija
* `index.html` – Početna stranica.
* `sadrzaj.html` – Katalog proizvoda/opreme.
* `kontakti.html` – Kontakt forma i lokacija.
* `css/tptpstil.css` – Glavni CSS fajl.
* `tptpskripte.js` – Glavni JavaScript fajl.
* `slike/` – Folder sa logotipom i ostalim medijima.
